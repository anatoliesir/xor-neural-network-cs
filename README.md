# Neural Network C#

This is a XOR logic gate model that learns from zero to answer correctly to the binary number we give to it. It doesn't have an interface, only console.
This is not like a python neural network, where everything is prepared. It is made in C#, and every detail was made from scratch. The most important thing to know, is that it was made 90% from my hard work, and the other 10% with AI(The json save class).


## Core Architecture & Math
There are much more math formulas used in the learning process, but these are the most important:
* Activation functions: Sigmoid, PRelu
* Error measurement: MSE (Mean Squared Error)
* Backpropagation by correcting mistakes: derivative of the "Activation functions".
